在 GitHub 上注册一个新账户，或者登录到现有的账户。

在 GitHub 上创建一个新的存储库。要创建一个存储库，请执行以下操作：

点击“+”号图标，然后选择“New repository”选项。

在“Repository name”字段中，输入要创建的存储库的名称。在这个例子中，你应该输入“try”。

可以选择性地为存储库添加描述和公共/私有访问权限。

点击“Create repository”按钮来创建该存储库。

在本地计算机上下载并安装 Git，这是一个用于管理源代码的开源版本控制系统。

使用 Git 初始化本地存储库，并将其连接到 GitHub 存储库。要完成这些操作，请执行以下操作：

打开命令行界面（如果使用 Windows，请使用“cmd”；如果使用 Mac 或 Linux，请使用“Terminal”）。

使用 cd 命令浏览到要创建本地存储库的文件夹。例如：cd Documents/GitHub

输入以下命令，以初始化本地存储库：git init
